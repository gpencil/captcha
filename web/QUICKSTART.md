# 验证码 Web 测试平台 - 快速开始

## 🚀 快速启动

### 方法一：使用启动脚本（推荐）

```bash
cd captcha-web
./start.sh
```

脚本会自动：
1. 检查 Redis 连接
2. 生成测试图片
3. 编译项目
4. 启动服务

### 方法二：手动启动

```bash
# 1. 确保正在运行 Redis
redis-cli ping

# 2. 生成测试图片（可选）
go run generate_test_images.go

# 3. 编译并运行
go run main.go handlers.go
```

## 📱 访问测试

打开浏览器访问：**http://localhost:8080**

## 🎮 使用说明

### 1. 字符验证码
- 点击"字符验证码"按钮
- 点击"生成验证码"
- 输入看到的字符
- 点击"验证"

### 2. 图片选择验证码
- 点击"图片选择"按钮
- 点击"生成验证码"
- 根据问题点击选择图片
- 点击"验证"

### 3. 滑动验证码
- 点击"滑动验证"按钮
- 点击"生成验证码"
- 拖动滑块到合适位置
- 点击"验证"

## 📝 注意事项

1. **Redis 必须运行**：验证码需要 Redis 存储
2. **测试图片**：首次运行会自动生成简单测试图片
3. **验证码过期**：5分钟内需要完成验证
4. **一次性使用**：验证成功后验证码会删除
